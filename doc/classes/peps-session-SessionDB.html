SessionDB.php : 21

Gestion des sessions en DB.

Design Pattern Singleton. NECESSITE une table (nommée 'Session' par défaut, constante 'SESSION_TABLE' dans Cfg) avec 3 colonnes : 'sid' (CHAR(N), PK) : longueur N définie par la constante SID_LENGTH dans Cfg) 'data' (TEXT) 'dateSession' (DATETIME) 3 modes possibles (@see SessionMode) : PERSISTENT, HYBRID, ABSOLUTE.
