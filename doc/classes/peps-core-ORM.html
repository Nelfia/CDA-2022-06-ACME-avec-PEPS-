ORM.php : 16

Abstraction ORM de la persistance des entités.

Indépentante du type de système de stockage. Si une propriété 'trucChose' est inaccessible, la méthode 'getTrucChose()' sera invoquée si elle existe. Sinon, null sera retourné. Implémente JsonSerializable pour préciser les propriétés NON PUBLIC à sérialiser. Ici, par défaut, toutes les propriétés.

Méthode appelée automatiquement par json_encode().

public jsonSerialize() : array<string|int, mixed>

Par défaut, seules les propriétés PUBLIC D'INSTANCE sont sérialisées. Avec cette méthode par défaut, les propriétés PUBLIC ET PROTECTED seront sérialisées en JSON. Elle les retourne sous la forme d'un tableau associatif comme attendu par json_encode(). Elle peut être redéfinie dans les classes entités si nécessaire.
